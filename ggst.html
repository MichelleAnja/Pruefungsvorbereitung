Grundstückgewinnsteuer

Die Grundstückgewinnsteuer ist eine kantonale Steuer. Beim Verkauf einer Liegenschaft fällt je nach Kanton auf den Gewinn die Grundstückgewinnsteuer an.

Wird eine Liegenschaft von einer natürlichen Person aus dem Privatvermögen veräussert, fällt darauf nur die Grundstückgewinnsteuer (keine Einkommenssteuer) an. Die Berechnung dieser Steuer ist bei im Privatvermögen gehaltener Liegenschaft in jedem Kanton gleich:

Erwerbskosten
+ Handänderungskosten beim Erwerb
+ Grundbuchgebühren beim Erwerb
= Erwerbswert
Verkaufserlös
− Erwerbswert
= Veräusserungsgewinnwert
− wertvermehrende Kosten
− Bemühungen zum Verkauf (Makler, Inserate)
= Veräusserungsgewinn

Für die Berechnung der Grundstückgewinnsteuer spielt die Haltedauer eine massgebende Rolle.

Grundstückgewinnsteuer (JP + NP im Geschäftsvermögen)

Die Abwicklung bei Veräusserung aus dem Geschäftsvermögen erfolgt je nach Kanton mit unterschiedlichen Verfahren. Auf den Gewinn ist zusätzlich die Gewinnsteuer nach DBG/DBSt zu zahlen:

Veräusserungspreis
− Veräusserungskosten
= Veräusserungswert
− Buchwert
= realisierter Buchgewinn
− 10% AHV (bei natürlichen Personen)
= steuerbarer Gewinn DBSt

Monistisches System (z. B. Zürich)

Die Grundstückgewinnsteuer wird auf folgendem Betrag erhoben:

Veräusserungspreis
− Veräusserungskosten
= Veräusserungswert
− Anlagewert
= Wertzuwachsgewinn

Die kantonale Gewinnsteuer wird auf folgendem Betrag erhoben:

Anlagewert
− Buchwert
= wiedereingebrachte Abschreibungen
− 10% AHV von der Berechnungsgrundlage Gewinnsteuer DBSt

Dualistisches System (z. B. Luzern)

Wie bei der DBSt über die Gewinnsteuer:

Erwerbskosten
+ Handänderungskosten beim Erwerb
+ Grundbuchgebühren beim Erwerb
= Erwerbswert
Verkaufserlös
− Erwerbswert
= Veräusserungsgewinnwert
− wertvermehrende Kosten
− Bemühungen zum Verkauf (Makler, Inserate)
= Veräusserungsgewinn

Wirtschaftliche Handänderung (Liegenschafts-AG)

Werden Aktien einer AG, die hauptsächlich Liegenschaften hält, veräussert, kann auf dem Veräusserungsgewinn eine Grundstückgewinnsteuer anfallen (wirtschaftliche Handänderung). Je nach Haltedauer, Art der Beteiligung (Privat- vs. Geschäftsvermögen) und Kanton erfolgt die Besteuerung unterschiedlich.

Werden die Aktien im Privatvermögen gehalten, ist vom Verkaufserlös das steuerfreie Eigenkapital sowie die stillen Reserven abzuziehen. Vom Gewinn können wertvermehrende Kosten in Abzug gebracht werden.
